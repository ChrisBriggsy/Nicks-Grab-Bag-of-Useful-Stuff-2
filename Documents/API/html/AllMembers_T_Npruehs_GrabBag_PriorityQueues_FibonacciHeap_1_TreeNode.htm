FibonacciHeap T  TreeNode MembersNick's Grab Bag of Useful Stuff 2.0 - Documentation

Implementation of a Fibonacci heap (abbreviated F-heap) by Michael L. Fredman and Robert Endre Tarjan which represents a very fast priority queue.

Provides insertion, finding the minimum, melding and decreasing keys in constant amortized time, and deleting from an n-item heap in O(log n) amortized time.
